# Technology Stack for Task Genie

This document outlines the core technologies and frameworks used in the Task Genie project.

## 1. Frontend

*   **Framework:** Next.js (React)
*   **Language:** TypeScript
*   **UI Library:** ShadCN UI
*   **Styling:** Tailwind CSS
*   **Cloud Integration & CI/CD:** AWS Amplify

## 2. Backend (Serverless)

*   **Cloud Provider:** Amazon Web Services (AWS)
    *   **Core Services:** AWS Lambda, AWS Step Functions, Amazon Bedrock (for AI/LLM integration), Amazon S3 (for storage, including Knowledge Bases)
*   **Language:** TypeScript
    *   **Runtime:** Node.js

## 3. Infrastructure

*   **Infrastructure as Code (IaC):** AWS Cloud Development Kit (CDK)
*   **Language for IaC:** TypeScript

## 4. CI/CD

*   **Backend CI/CD:** GitHub Actions

## 5. Documentation

*   **Framework:** Docusaurus
